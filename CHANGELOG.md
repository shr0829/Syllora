# 更新日志

## v0.1.0 - 2026-04-26

### 版本说明

这是 Syllora 的首个完整可运行版本，重点完成了从“用户输入学习主题”到“研究、规划、阶段讲解、多图图示、结构化落盘、前端实时展示”的整条链路。

### 品牌与仓库更新

- 项目品牌统一为 **Syllora**
- GitHub 仓库名称统一为 **Syllora**
- 新增 GitHub 首页封面图，强化公开仓库展示效果

### 新增功能

#### 1. 学习主题项目化

- 支持把每个学习主题创建为一个独立项目
- 为每个项目保存对话、研究文档、学习计划、阶段详情与图片资产
- 支持本地文件化仓库结构，便于追踪与二次处理

#### 2. 真实文本模型接入

- 支持 OpenAI-compatible 文本接口
- 支持 `responses` 与 `chat/completions` 两种调用路径
- 支持联网研究阶段调用 web search 能力
- 支持模型参数从 `config/ai.config.toml` 统一读取

#### 3. 真实生图模型接入

- 接入 `gpt-image-2`
- 支持多图片渠道配置
- 支持主渠道失败时自动回退到备用渠道
- 渠道都失败时自动回退到本地占位 SVG，保证流程不中断

#### 4. 阶段化学习路径生成

- 自动把研究结果拆成多个学习阶段
- 每个阶段包含摘要、目标、动作、产出、预计时长等结构
- 前端支持阶段概览与阶段详情跳转

#### 5. 阶段知识讲解生成

- 点击阶段即可生成当前阶段的详细知识讲解
- 输出为稳定可解析 Markdown
- 自动解析并落盘为 JSON 结构，供前端稳定展示

#### 6. 多图生成

- 每个阶段默认生成 2 张讲解图
- 图示类型包括：
  - 阶段结构总览
  - 阶段学习流程
- 前端支持图片内联展示、点击放大与阶段详情联动

#### 7. 并发生成能力

- 支持多个阶段并发生成
- 支持单阶段内部多图并发生成
- 新增批量接口：
  - `POST /api/projects/:id/lessons/batch`

#### 8. 流式更新与实时刷新

- 支持生成过程显式进度展示
- 支持分段流式更新正文草稿
- 支持项目页与阶段页统一进度体验
- 前端通过 SSE 订阅项目事件流，自动刷新生成状态

#### 9. 公式支持

- 约束模型使用：
  - 行内公式：`$...$`
  - 块公式：`$$...$$`
- 前端接入 MathJax
- 支持在阶段详情中渲染公式，而不是原始文本裸显示

#### 10. 输出稳定性与解析链路

- 强化 prompt 约束，减少自由格式漂移
- 研究 / 计划 / 阶段讲解都经过解析与规范化
- 统一走：

```text
generate → parse → normalize → persist
```

### 前端体验更新

- 首页布局向更简洁的学习产品风格收束
- 阶段卡片支持实时草稿 / 阶段摘录展示
- 批量生成时支持全局进度与阶段状态联动
- 阶段详情页支持：
  - 多图展示
  - 公式展示
  - 文本与图示统一阅读流

### 配置能力

- 新增：
  - `config/ai.config.toml`
  - `config/ai.config.template.toml`
- 支持文本模型配置与图片模型配置分离
- 支持多图片渠道模板配置

### 验证结果

- 后端编译通过
- 前端构建通过
- 本地健康检查通过
- 真实项目已完成研究、计划、批量阶段生成验证
